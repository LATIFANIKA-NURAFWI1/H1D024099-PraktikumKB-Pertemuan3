## 📄 Deskripsi File
- **petshop.py** → Program fuzzy untuk menentukan jumlah stok makanan hewan  
- **pelayanan.py** → Program fuzzy untuk menentukan tingkat kepuasan pelayanan masyarakat  

- Python
- NumPy
- scikit-fuzzy
- Matplotlib

---

##  Studi Kasus 1: Stok Pet Shop

### Input:
- Barang Terjual
- Permintaan
- Harga per Item
- Profit

### Output:
- Rekomendasi Stok Makanan

### Tujuan:
Menentukan jumlah stok makanan berdasarkan kondisi penjualan dan keuntungan.

---

## Studi Kasus 2: Kepuasan Pelayanan

### Input:
- Kejelasan Informasi
- Kejelasan Persyaratan
- Kemampuan Petugas
- Ketersediaan Sarpras

### Output:
- Tingkat Kepuasan Pelayanan (0–400)

### Tujuan:
Menilai kualitas pelayanan berdasarkan beberapa aspek.
